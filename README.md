### Montagem de um Computador <h3>
  
O projeto escolhido foi a montagem de um Computador. 
Para realizar a montagem será necessário os seguintes materiais:

* Placa mãe;
* Disco rígido;
* Memória RAM;
* CPU;
* Cooler da CPU;
* DVD-ROM;
* Fonte de energia;
* Gabinete;
* Chave de fenda;
* Sistema operacional.
